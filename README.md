# Machine-learning---supervised-learning-classification---gaussian-naive-bayes-breast-cancer-wisconsin
This notebook performs binary classification on the Breast Cancer Wisconsin (Diagnostic) dataset included with scikit-learn (malignant vs benign).
It follows a step-by-step ML workflow: load data, inspect dataset, train/test split, train a Gaussian Naive Bayes model, evaluate with accuracy, confusion matrix + heatmap, and classification report, then run 10-fold cross-validation.
Finally, it compares multiple classifiers (GaussianNB, KNN, LogisticRegression, SVC) and reports which model performs best by mean CV accuracy.
